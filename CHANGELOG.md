## v1.1 — 2026-06-21

### 추가
- **스마트 갱신 스케줄**: 평일 주간(08~16시) 1h / 야간 2h / 주말·공휴일 6h 간격 자동 전환
- **한국 공휴일 처리**: 2026~2027년 공휴일에 6시간 주기 적용 (lunar 기반 공휴일은 매년 수동 업데이트)
- **다음 갱신 시각 표시**: 헤더에 "다음 갱신: HH:00 (N분 후 · 주간/야간 Xh 간격)" 실시간 표시
- **새로고침 버튼**: 헤더에 🔄 버튼 추가 — 클릭 시 최신 배포 데이터로 페이지 즉시 갱신
- **탭 가로 스크롤**: 10개 탭이 좁은 화면에서 가로 스크롤로 접근 가능 (잘림 해소)
- **배당달력 오늘 날짜 강조**: 오늘 날짜에 파란 원 + 파란 링 표시, 요일 색상 충돌 수정
- **ISA 비과세 한도 프로그레스 바**: 절세분석 탭 ISA 카드에 한도 사용률 시각화 + 잔여 금액 표시

### 변경
- `kakao_watcher.py`: `_hourly_price_refresh_loop()` → 스마트 스케줄 (`_next_refresh_seconds()` 함수 분리)
- `stock-dashboard.html`: APP_VERSION 상수 도입 (`const APP_VERSION = 'v1.1'`)

## v1.0 — 2026-06-19 (베이스라인 정의)

> 이전 개발 이력을 v1.0으로 통합 정의 (2026-06-21 소급 적용)

### 포함된 기능
- **대시보드 탭**: 총 평가금액 요약 카드, FIRE 진행률, 배당수익률, 수익률 차트
- **포트폴리오 탭**: 7개 계좌 종목별 현재가·수익률·평가손익 테이블
- **수익률 탭**: 계좌별 수익률 히트맵, 평가금액 추이 차트
- **배당금 탭**: 배당 이력, 배당수익률 차트
- **당일 거래 탭**: 당일 매수·매도 내역
- **계좌비교 탭**: 7개 계좌 누적 수익률·배당 비교
- **절세분석 탭**: 일반/ISA/연금/IRP 배당세 분석, 종합과세 헤드룸, 절세 추천
- **배당달력 탭**: 월별 캘린더 배당 시각화, 연간 막대 그래프
- **파일 불러오기 탭**: 카카오톡 CSV 드래그앤드롭 + 자동 파싱
- **관리 탭**: 환율·FIRE 목표 설정, JSON 내보내기/가져오기
- **카카오 워처**: 카카오톡 Google Drive 자동 감지 → 파싱 → GitHub Pages 자동 배포
- **FIRE 목표 기본값**: 10억원 (localStorage 우선 → watcher 기본값 → 0 순위)
- **현재가 자동 갱신**: 매 정시 yfinance + pykrx 조회 후 GitHub Pages push
