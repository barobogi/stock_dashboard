> 생성일: 2026-06-20  
> 대상: stock-dashboard 프로젝트  
> 분석 기준: 현재 기능 검토 + 글로벌 포트폴리오 관리 도구 벤치마킹

---

## 📊 현재 프로젝트의 강점

### ✅ 자동화 및 실시간성
- **카카오톡 기반 자동 파싱**: 수동 입력 없이 삼성증권 알림 자동 수집
- **매 정시 현재가 갱신**: 백그라운드 스레드로 지속적인 가격 업데이트
- **GitHub + GitHub Pages 자동 배포**: push → 2~3분 내 라이브 반영 (Netlify → GitHub Pages 전환, 크레딧 초과 이슈)

### ✅ 개인화 및 유연성
- **다중 계좌 추적**: 7개 계좌 (일반, ISA, 퇴직연금, 연금저축)
- **포트폴리오 기준일 설정**: 자체 규칙(기준일 12:50 기준) 적용 가능
- **localStorage 기반 수동 편집**: 사용자가 직접 조정 가능

### ✅ 데이터 기반 인사이트
- **배당금 추적**: 세금 구분(15.4%/9.9%/16.5%) 및 월평균 계산
- **FIRE 진행도**: 목표 금액 대비 달성률 + 배당 기준 예상 소요 연수
- **손익분기점(BEP)**: 포트폴리오 테이블에 표시

## 🔍 현재 프로젝트의 한계

### ⚠️ 기능 측면
1. **시각화 제한**
   - 포트폴리오 구성: 원형 차트만 제공 (계좌별, 종목별)
   - 성과 분석: 누적 수익만 표시, 기간별 추이 없음
   - **부재**: 히트맵, 위험도 분석, 변동성 추적

2. **데이터 부족**
   - 스냅샷 이력 7일 미만 (기간별 수익률 그래프 미지원)
   - 거래 비용 추적 없음 (수수료, 세금 상세 분석 부족)
   - 섹터/업종별 자산 배분 분석 없음

3. **예측 기능 없음**
   - 배당 예측: 최근 6개월 단순 평균 (계절성 미고려)
   - 목표 달성 일정: FIRE 연수만 표시, 구체적 마일스톤 없음

### ⚠️ 기술 측면
1. **인증/보안**
   - 이메일 발송 인증 미해결 (네이버 2FA 이슈)
   - 클라우드 백업 체계 없음 (로컬 폴더/GitHub만 의존)

2. **에러 처리**
   - yfinance NumPy 충돌 (간헐적 폴백으로 관리 중)
   - kakao_watcher 비정상 종료 시 자동 재시작 로직 없음

3. **확장성**
   - CSV/Excel 내보내기 없음
   - API 제공 안 함 (다른 도구 연동 불가)

## 💡 단계별 개선안

### 🟦 Phase 1 (즉시 구현, 1~2주) — 핵심 시각화 강화

#### 1.1 포트폴리오 히트맵 추가
```
기능: 종목별 수익률 히트맵 (Finviz 스타일)
- 타일 크기: 현재 평가금액
- 타일 색상: 수익률 (녹색/빨강색 그래디언트)
- 호버: 수익률%, 수익금액, 배당금
- 클릭: 해당 종목 상세 정보

기대효과:
  ✓ 포트폴리오 전체 성과를 한눈에 파악
  ✓ 강세 종목/약세 종목 즉시 식별
  ✓ 자산 배분 시각화 (웅큼)
```

#### 1.2 기간별 수익률 그래프 추가
```
기능: 일/주/월/연 수익률 트렌드
- 스냅샷 데이터 기반 (현재 보존 중)
- 일간/누적 손익 이중 Y축
- 주요 이벤트 마커 (배당일, 거래일)

구현:
  📊 Chart.js 또는 Recharts 활용
  📅 localStorage 스냅샷 7일 이상 누적 시작
  
기대효과:
  ✓ 포트폴리오 성과 추이 추적
  ✓ 목표 달성 진척도 시각화
  ✓ 시기별 운용 성과 비교
```

#### 1.3 계좌별 비교 대시보드
```
기능: 7개 계좌의 성과를 한 화면에서 비교
- 계좌별 수익률, 배당금, ROI 비교
- 계좌별 자산 배분 비교
- 계좌별 우수/열화 종목 TOP3

기대효과:
  ✓ 계좌별 운용 성과 평가
  ✓ 최고 수익 계좌 전략 분석
```

---

### 🟦 Phase 2 (1개월) — 기능 확대

#### 2.1 위험도 분석 추가
```
기능: 포트폴리오 위험 지표
- 변동성(Volatility): 종목별 추이 변동성 계산
- 베타(Beta): 시장 대비 민감도
- 샤프 지수(Sharpe Ratio): 위험 조정 수익률
- 상관관계: 종목 간 연동성 시각화

데이터 소스:
  📈 yfinance 히스토리 데이터 활용
  📊 KOSPI 200 수익률과 비교 (베타 계산)

기대효과:
  ✓ 포트폴리오 위험 수준 정량화
  ✓ 다각화 효율성 평가
```

#### 2.2 세금 최적화 도구
```
기능: 절세 시나리오 계산
- 매도 순서 제안 (FIFO, LIFO, 평균단가)
- 손실 보전(Harvesting) 기회 식별
- 연간 예상 세금 계산
- 손실 이월 추적 (연금계좌 제외)

기대효과:
  ✓ 연간 절세액 예상 (평균 5~15%)
  ✓ 매도 전략 수립 근거 제공
```

#### 2.3 배당 일정 관리
```
기능: 배당금 달력 및 예측
- 각 종목의 배당 주기 (월/분기/연간)
- 다음 배당일 자동 예측
- 배당금 현금화 타임라인
- 배당금 재투자 시뮬레이션

기대효과:
  ✓ 배당금 수령 계획 수립
  ✓ 유동성 관리 개선
```

### 🟦 Phase 3 (2~3개월) — 고급 기능

#### 3.1 AI 기반 포트폴리오 리밸런싱 제안
```
기능: 자동 자산 배분 최적화
- 현재 포트폴리오 vs 목표 배분 비교
- 리밸런싱 거래 제안 (매매 건수 최소화)
- 예상 수익 변화 시뮬레이션
- 거래 비용 차감 후 순이득 계산

기대효과:
  ✓ 자산 배분 최적화로 수익 개선
  ✓ 거래 빈도 감소 (세금/수수료 절감)
```

#### 3.2 모바일 앱 (PWA) 지원
```
기능: 반응형 디자인 + PWA
- 오프라인 작동 지원
- 홈 화면 설치 가능
- Push 알림 (배당일, 목표 달성 등)

기대효과:
  ✓ 언제 어디서나 포트폴리오 확인
  ✓ 즉각적인 거래 알림
```

#### 3.3 데이터 내보내기 & 통합
```
기능: 다양한 포맷 지원
- CSV/Excel 내보내기
- Google Sheets 연동
- Broker API 직접 연동 (삼성증권, 한국투자증권)

기대효과:
  ✓ 다른 도구와 상호 운용성
  ✓ 데이터 포팅 용이
```
